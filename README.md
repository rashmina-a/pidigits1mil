# PiDigit1Mil benchmark
This python program can be ran on any device that can run python and it can benchmark your device and give the time it took to generate the first 1 million digits of Pi.For comparison on my laptop with **16GB RAM**, **Ryzen 5 3500U CPU**, and **Vega 8 integrated graphics** It took **22.88** seconds :-

# How to install
Run this command 

`git clone https://github.com/rashmina-a/pidigits1mil.git && cd pidigits1mil && pip install -r requirements.txt && python3 gen.py`

# Usage
This program is mostly useful for stuff like Benchmarking and for math nerds 🤓
# How it works
Becuase 1,000,000 is a huge number i also added a math library called mpmath to to calculate the digits correctly . 
# Troubleshooting
If you encounter any commands returning the error to create a python virtual environment search it up on google or run the following command 

`python3 -m venv venv && source venv/bin/activate`

Replace python with the suitable version if your from the future ⌚️
